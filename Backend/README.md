# Invoicing System Backend

Please refer to the root [README.md](../README.md) for full architecture design, data modeling rationale, API endpoints documentation, and setup instructions.

### Quick Start
1. Install dependencies:
   ```bash
   npm install
   ```
2. Seed the database:
   ```bash
   node seed.js
   ```
3. Start the server:
   ```bash
   node server.js
   ```
